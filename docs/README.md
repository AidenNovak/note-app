# Docs

这部分放仓库级说明文档，不放产品设计稿，也不放各子项目自己的内部文档。

## 仓库入口

- `repository-layout.md` — 顶层目录说明，帮助快速判断文件应该放哪里
- `local-development.md` — 本地启动、连接、测试的统一说明

本地产物统一放在仓库根目录的 `_local/`，不放在 `docs/`。

## 其他文档位置

- `../design-docs/` — 产品和设计相关文档、原型、评审记录
- `../backend/docs/` — 后端部署、基础设施和后端内部说明
- `../easystarter/docs/` — Native 工作区自己的说明
